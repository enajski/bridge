# Bridge Example Artifacts

These files are illustrative examples of Bridge artifact shapes and vocabularies.

Current examples emphasize:

- `verification-brief`
- `observable-contract`
- `differential-evidence-card`
- evaluation-profile and evaluation-report shapes for benchmark slices such as SysMoBench
- status vocabularies such as:
  - `expected-window`
  - `known-deviation`
  - `spec-unfaithful`
  - `clean-with-scope`
- richer system categories such as:
  - `actor-message`
  - `lock-free`
  - `async-runtime`
